Простой проект на языке Kotlin с использованием библиотеки JavaFX и визуальным построением разметки главного окна в Scene Builder. Настроено создание исполняемого файла (Build Artifacts). Создан типовой класс контроллера окна, и на его основе – классы главного и дополнительного окон приложения. Для работы требуется Java 1.8.
